## LCSTS_new

- LCSTS_new中文短摘要生成数据集：https://www.luge.ai/#/luge/dataDetail?id=10

生成式短摘要数据集，以微博原文为输入，1~2句话的短摘要为输出


LCSTS_new是中文短摘要最常用的LCSTS短摘要数据集的升级版本，在数据量、质量方面均有显著提升，在信息摘要与提炼的过程中，与原文的事实一致性需要得到重点关注。
